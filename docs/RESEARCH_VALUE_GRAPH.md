# MEMBRA Research Value Graph

## Thesis

Research papers can accumulate economic value when software ecosystems, tooling, and infrastructure are built on top of them.

## Example chains

| Research | Repositories | Infrastructure Outcome |
|---|---|---|
| Attention Is All You Need | transformers, PyTorch, Ollama | LLM ecosystems and automation |
| MapReduce | Hadoop, Spark, Kubernetes | Distributed compute and cloud orchestration |
| Bitcoin Whitepaper | Bitcoin Core, Geth, Solana | Cryptographic settlement infrastructure |
| Raft Consensus | etcd, Consul, Kubernetes | Reliable distributed coordination |

## Pipeline

The GitHub Actions workflow runs a Python pipeline that exports a machine-readable research value graph.

## Boundary

This repository focuses on scientific, economic, and software infrastructure.
