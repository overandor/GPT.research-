## How To Interpret File Prices

A file's assigned price is not based only on file size. It reflects the file's role in buyer diligence:

1. **Executable files** support technical utility.
2. **Connector files** support data access and operational reach.
3. **Dossier files** support institutional interpretation.
4. **Evidence files** reduce buyer uncertainty.
5. **Transfer files** make the asset saleable.
6. **Market files** turn appraisal into buyer demand.

## Verification Rule

Use these labels when updating the ledger:

| Status | Meaning |
|---|---|
| Pending | File is needed but not yet attached |
| Attached | File exists in repository |
| Verified | File exists and evidence has been reviewed |
| Buyer-reviewed | File has been reviewed by a buyer, investor, or auditor |
| Transaction-backed | File contributed to an offer, LOI, escrow, or closed transaction |
